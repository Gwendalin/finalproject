Help Manual

About the System


This website was created to facilitate the creation of garments with high creativity and high-quality by fashion designers. Several functions are provided in the website that can be fully utilized by fashion designers for the easy process of garments creation without any extra costs.

Functions provided:

  • 2D design workspace
  • 3D design generation
  • Color Guidance
  • Style and Trends Recommendation
  • Upcoming Event Tracking (e.g., fashion event)

About 2D Design Working Space Functions


  • The 2D design workspace is used to create 2D designs.
  • A variety of tools are provided on the left sidebar for the creation of designs work, such as lines, polygons, shapes, etc.
  • A "Save" option is provided to save the work and a "Share" option is provided to share the work among friends.

About 3D Design Generation Functions


  • The 3D Design Generation is used to convert the 2D designs into 3D designs to make designs more dimensional and vivid.
  • Adjustments features are provided to enhance the designs' view.
  • Fashion designers are allowed to return to the 2D design working space for design editing.

About Color Guidance Functions


  • The color guidance feature will provide suggestions for color matching and extraction of colors.

About Style and Trends Recommendation Functions


  • This website provides style and fashion trend recommendations, keeping up with the latest styles and fashion trends, so that design works can keep up with real-time fashion trends.
  • Recommendations can be saved for reference.

About Upcoming Events Tracking Functions


  • The latest upcoming events such as fashion events will be tracked.
  • Reminder can be set for the upcoming events.

How to use the system?


Step 1: User has to Login/Register an account to perform tasks. The user can click on the Login/Register button provided on the top right side.

Step 2: After Logging in/Registering, the user is allowed to perform tasks by clicking on the navigation buttons provided at the top (2D Design, 3D Generation, Color Guidance & Upcoming Event).
